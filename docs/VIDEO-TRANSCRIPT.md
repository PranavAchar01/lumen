**[0:00 – title card]** This is Lumen — an accessibility agent that runs entirely at the edge, built and deployed on Tencent EdgeOne Makers.

**[0:10 – problem card]** Ninety-five percent of the top million homepages fail basic accessibility. For over 285 million people with visual impairment, most of the web is exhausting — or simply unusable. Accessibility overlays only help on sites that opt in. Lumen flips the model: it gives the reader an agent that repairs any page, on demand.

**[0:34 – LIVE screen recording of lumen.edgeone.cool]** A Wikipedia URL is typed in and "Make it readable" is clicked. Paste any link. Lumen fetches the page at the nearest of EdgeOne's 3,200 edge nodes, strips away the clutter, and rebuilds it with EdgeOne's built-in DeepSeek model. No backend. No API keys. *(The ~21-second model rebuild is shown at 10× speed, labeled on screen.)*

**[0:49 – live recording continues]** Here's a real page, rebuilt live. A plain-language summary. Clean sections a screen reader can navigate. A page map of the original headings. Key actions, pulled out of the noise. And you can ask the agent anything about the page — the recording shows the question "What are the main ways screen readers get information from the page?" answered faithfully from the page content (off-screen models vs. accessibility APIs). There's also built-in text-to-speech, high contrast mode, and a keyboard-first interface.

**[1:17 – stack card]** Everything runs on the sponsor's stack. Pages hosting. Edge functions. Built-in edge AI. And KV caching — so one reader's rebuilt page becomes every reader's instant page, worldwide.
