        Why this runs at the edge

        Edge-node fetch

        Pages are fetched and cleaned inside an EdgeOne edge function running on EdgeOne's global network — with an SSRF guard on every URL.

        Built-in edge AI

        Rewriting happens with EdgeOne’s built-in DeepSeek models, invoked directly inside the edge function. No keys, no backend.

        Shared KV cache

        Reading views are cached in EdgeOne KV (when enabled), so a repeat request is served without re-running the model.
